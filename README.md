# Labadmin Linux Tools
<img align="right" src="https://cdn4.iconfinder.com/data/icons/online-marketing-hand-drawn-vol-3/52/online__options__services__setting__gear__option__support-128.png">
Labadmin Win Tools is a collection of PowerShell scripts to admin Windows 10 systems in a lab school environment. 

## Install
```
cd /opt/
git clone "https://github.com/leomarcov/labadmin-linux-tools"
chmod a+x /opt/labadmin-linux-tools/labadmin-*
chmod 770 /opt/labadmin-linux-tools 
```
